 🚀 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Access the Page:
  • Navigate to: Sidebar → HRM → Attendance
  • Direct URL: /hrm-attendance
  • Page title: "Attendance V3.0"

Mark Attendance:
  1. Click "Mark Attendance" button (top-right)
  2. Select employee from dropdown
  3. Choose work mode (Office/Remote/Hybrid/Site)
  4. Optional: Add location & notes
  5. Click "Check In"

Filter Records:
  • Search: Type employee name or ID
  • Department: Select from dropdown
  • Status: Choose status type
  • Work Mode: Select mode
  • Date Range: Set start and end dates
  • Calendar: Click date to filter
  • Late Only: Check for late employees

Bulk Operations:
  1. Check multiple record checkboxes
  2. Click "Mark Present (N)" or "Delete (N)"
  3. Confirm action

Export Data:
  • Click "Excel", "CSV", or "PDF" button
  • File downloads automatically
  • Format: attendance-YYYY-MM-DD.ext

Calendar Interaction:
  • Click date → Table filters to that date
  • Click again → Remove filter
  • Green = Present, Red = Absent, Yellow = Late
  • Blue dots = WFH employees
