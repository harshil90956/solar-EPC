   └─ Or direct: /hrm-attendance

2. Mark Attendance
   └─ Click "Mark Attendance" button
   └─ Select employee
   └─ Choose work mode (Office/Remote/Hybrid/Site)
   └─ Click "Check In"

   └─ Search: Type employee name
   └─ Department: Select from dropdown
   └─ Status: Choose status type
   └─ Calendar: Click date to filter

   └─ Check multiple records
   └─ Click "Mark Present (N)" or "Delete (N)"

5. Export Data
   └─ Click "Excel", "CSV", or "PDF" button
   └─ File downloads automatically
