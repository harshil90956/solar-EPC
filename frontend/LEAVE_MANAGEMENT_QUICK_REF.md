## 🔍 Quick Actions

### Apply Leave
1. Click **"Apply Leave"** button
2. Select employee & leave type
3. Choose dates
4. Enter reason
5. Submit

### Approve Leave
1. Find pending leave in table
2. Click **"Approve"** button
3. Confirm approval
✅ Done!

### Reject Leave
1. Find pending leave
2. Click **"Reject"** button
3. Enter rejection reason
4. Confirm
❌ Rejected!

### View Details
1. Click **"View"** button
2. See complete information
3. Take action if needed

## 🗓️ Calendar Usage

### Select Date
- **Click any date** → Filter leaves for that day
- **Green dot** (🟢) = Approved leaves
- **Amber dot** (🟡) = Pending leaves
- **Today** = Blue highlighted
- **Selected** = Primary ring

### Navigate
- **← Button** = Previous month
- **→ Button** = Next month

---

## 🔎 Filtering

### Search
Type in search box:
- Employee name (John Doe)
- Employee ID (EMP001)

### Filters
- **Status**: All / Pending / Approved / Rejected
- **Type**: 9 leave types available
- **Date Range**: Start + End dates

### Combined
All filters work together!

## 🚦 Common Workflows

### Manager Approving Leaves
```
1. Open Leave Management
2. Filter by "Pending"
3. Review each request
4. Click "View" for details
5. Click "Approve" or "Reject"
6. Done! ✅
```

### Employee Checking Leave Status
```
1. Open Leave Management
2. Search for own name
3. View status in table
4. Click "View" for details
5. Check approval info
```

### HR Analyzing Leave Patterns
```
1. Open calendar panel
2. Navigate through months
3. Click dates with many leaves
4. Review patterns
5. Export data (if available)
```

## 🐛 Troubleshooting

### Issue: Calendar not showing leaves
**Fix**: Refresh page, check date filters

### Issue: Can't approve leave
**Check**: 
- You have permission
- Leave is in "Pending" status
- Network connection

### Issue: Search not working
**Try**:
- Clear search box
- Refresh page
- Check spelling

## 📞 Quick Help

### Can't find a leave?
→ Use search or date filters

### Need to edit a leave?

### Want to see all leaves for a date?
